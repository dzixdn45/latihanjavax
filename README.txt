Cara menggunakan applikasi bengkelku

applikasi ini masih rebuild artinya belum bisa digunakan secara instan hanya untuk di netbeans saja
cara menggunakanya adalah

1. buka netbeans
2. pilih file klik open project lalu pilih file yang udah didownload 
3.tunggu akan muncul projectnya 

lalu untuk impor databasenya adalah

1. nyalakan server apache dan mysql di xampp
2. buka browser lalu klik link localhost/phpmyadmin
3. buat database baru dengan nama bengkelku
4. setelah buat lalu klik impor 
5. lalu choose database plih bengkelku(1).sql
6. lalu impor
